Platform Overview

GraphR represents a revolutionary approach to mathematics education by leveraging mobile devices as powerful learning tools rather than restricting their use. This platform embodies first principles thinking in education: instead of banning phones in classrooms, GraphR transforms them into controlled, secure learning instruments.

Mission: Enable educators and students to use smartphone technology productively while maintaining academic integrity and preventing cheating through innovative exam monitoring.

Problem Solved

Many school districts (particularly in Ohio under recent education reform initiatives) are implementing phone bans and Faraday pouches. These approaches:

  • Endanger student safety by preventing emergency communication
  • Reduce educational access to powerful computational tools
  • Create behavioral friction rather than solving underlying problems

GraphR Solution

GraphR provides a secure platform that:

  • Uses patent-pending Test Mode to prevent cheating without locking devices
  • Detects unapproved app switching and reports violations to teachers
  • Provides powerful calculators and graphing tools for learning
  • Maintains open communication channels for emergencies
  • Delivers real-time analytics on student performance

Exam Monitoring Features

  • Records complete activity log for each exam session

Data Flow

Student Device |-- Authentication --> Server |-- Exam Data Sync --> Server Cache |-- Real-time Activities --> Activity Queue |-- Unapproved Actions --> Alert System |-- Exam Submission --> Grading Server |-- Analytics --> Teacher Dashboard

Key Design Principles

  • Offline-First: App functions without connectivity, syncs when network available
  • Security First: All data encrypted in transit and at rest
  • Real-Time Monitoring: Instant detection and reporting of violations
  • Responsive Design: Works on phones, tablets, and web browsers
  • Scalable: Handles thousands of concurrent exam sessions

Patent Technology Overview

GraphR is built on proprietary exam monitoring technology covered by two US patents.

US Patent 10,339,827 B2

Title: Computer-Implemented System and Method for Administering an Examination

Filed: November 1, 2016 | Issued: July 2, 2019

This patent covers the core technology for:

  • Mobile device-based examination administration
  • Real-time detection of unapproved activities on student devices
  • Server-side transmission of activity notifications
  • Interactive calculator parameter customization
  • Multi-function calculator interface for testing

US Patent 10,839,708 B2

Title: Computer-Implemented System and Method for Administering an Examination

Filed: May 8, 2019 | Issued: November 17, 2020

This continuation patent extends coverage to:

  • Enhanced activity monitoring and logging
  • Improved unapproved activity detection algorithms
  • Advanced exam session management
  • Teacher-configurable exam restrictions
  • Real-time student activity dashboard
Patent Innovation: Unlike lockdown browsers that disable device functionality, GraphR allows students to use their phones normally while monitoring for violations. This preserves access to emergency features while preventing cheating.
