# GraphR Launch Plan

Exam proctoring calculator app. Free calculator, paid proctoring subscription. iOS + Android via React Native/Expo.

## Phase 1: App Store Ready

- [ ] EAS production builds for iOS and Android
- [ ] Firebase production project (separate from dev)
- [ ] Firestore security rules locked down
- [ ] In-app purchase flow tested end to end
- [ ] COPPA compliance check
- [ ] Accessibility audit (VoiceOver, TalkBack)
- [ ] Load test with 30+ concurrent exam takers
- [ ] App Store screenshots and descriptions
- [ ] Privacy policy and ToS on graphrapp.com

## Phase 2: Beta Testing

- [ ] TestFlight + Google Play internal track
- [ ] Get 5-10 teachers to run real exams with it
- [ ] Confirm violation detection works reliably
- [ ] Teacher dashboard with 30 students
- [ ] Push notifications working
- [ ] Collect teacher feedback, fix top issues

## Phase 3: Submit to Stores

- [ ] Apple App Review (2-4 weeks for education)
- [ ] Google Play Review
- [ ] Handle any rejections
- [ ] Support email and FAQ page

## Phase 4: School Pilots

- [ ] 3-5 schools, free premium access
- [ ] Teacher onboarding materials
- [ ] Track usage: exams created, violations caught
- [ ] Get feedback, iterate
- [ ] Explore district-level licensing

## Phase 5: Launch and Grow

- [ ] Ed-tech forums and social media
- [ ] Conference presence (ISTE, FETC)
- [ ] Teacher referral program
- [ ] LMS integrations (Google Classroom, Canvas, Schoology)
- [ ] University market

## Targets

| | Pilot | 3 Month | 12 Month |
|-|-------|---------|----------|
| Schools | 5 | 25 | 200 |
| Active teachers | 15 | 100 | 1,000 |
| Exams run | 50 | 500 | 10,000 |
