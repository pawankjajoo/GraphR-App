# GraphR App

Educational calculator and exam proctoring app built with React Native/Expo.

## Core Features

- Multi-mode scientific calculator (Basic/Scientific/Graphing)
- Secure exam proctoring with app-switch detection
- Classroom management and student enrollment
- Real-time grading and performance analytics
- Role-based access (Student/Teacher/Admin)

## Subscriptions

- Free, Pro, School tiers via expo-in-app-purchases

## Architecture

- 12 screens, 7 service modules
- Web platform: standalone HTML/JS version with graphing, calculator, classroom, and exam modules
- GraphR_Demo.html - self-contained web demo of the calculator
- Web app with 7 JS modules and 4 CSS files
- Archive materials from 2016-2021 (original web app, pitch decks, demo videos)

## Tech Stack

- React Native, Expo SDK
- Firebase (Auth with email + Google SSO, Firestore, Storage)
- Chart.js (web graphing)
- Canvas API (web graphing)

## Status

Development - architecture complete, Firebase config needs setup, some features use demo data
